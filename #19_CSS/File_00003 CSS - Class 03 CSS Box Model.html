Margin and Padding

Margin creates space outside an element, while padding creates space inside an element.

Border Properties

Border adds a visual border around an element. Border properties include style, width, color, and radius.

Width and Height

Width and height set the dimensions of an element's content box.

Box Sizing

Box sizing affects how width and height properties are calculated, including padding and border.

Margin Collapse

Margin collapse occurs when adjacent margins of elements combine, creating a smaller margin.
