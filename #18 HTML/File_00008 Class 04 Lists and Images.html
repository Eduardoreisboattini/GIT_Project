      • Ordered lists are similar to unordered lists, but they have a specific order indicated by numbers.

      <ol> <li>First Item</li> <li>Second Item</li> </ol>
    • Creating Definition Lists

      • Definition lists are used to define terms and their corresponding descriptions.

      <dl> <dt>Term 1</dt> <dd>Description 1</dd> <dt>Term 2</dt> <dd>Description 2</dd> </dl>
    • Inserting Images

      • Images can be inserted using the `` element with the `src` attribute pointing to the image file.

      <img src="image.jpg" alt="Image">
    • Image Attributes and Alt Text

      • Alt text is important for accessibility and should be included with images using the `alt` attribute.

      <img src="image.jpg" alt="Description of the image">
