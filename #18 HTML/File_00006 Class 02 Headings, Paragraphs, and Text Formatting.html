Class 2: Headings, Paragraphs, and Text Formatting

Topic 1: Using Headings (h1-h6)

Headings are used to define the hierarchical structure of your content. They range from h1 (highest importance) to h6 (lowest importance).

  • <h1> Heading Level 1</h1>

    Results in:

    Heading Level 1

  • <h2> Heading Level 2</h2>

    Results in:

    Heading Level 2

  • <h3> Heading Level 3</h3>

    Results in:

    Heading Level 3

  • <h4> Heading Level 4</h4>

    Results in:

    Heading Level 4

  • <h5> Heading Level 5</h5>

    Results in:

    Heading Level 5

    Topic 2: Creating Paragraphs

    Paragraphs are used to group and structure blocks of text. Each paragraph is represented by the <p> tag.

    Topic 3: Text Formatting (Bold, Italic, Underline)

    You can apply various text formatting styles to your content:

    • Bold: Use the <strong> or <b> tag.
    • Italic: Use the <em> or <i> tag.
    • Underline: Use the <u> tag.

    Topic 4: Line Breaks and Horizontal Rules

    You can add line breaks using the <br> tag, and horizontal rules using the <hr> tag.

    Topic 5: Adding Comments in HTML

    Comments are used to add notes to your code that won't be displayed in the browser. Use <!-- ... --> to create comments.
