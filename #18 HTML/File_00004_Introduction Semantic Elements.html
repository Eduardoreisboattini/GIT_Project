Topic 1: Introduction to Semantic Elements

  • <header> It's used to define the header of a page or a section, typically containing introductory content, headings, logos, and navigation menus.
  • <nav> Represents a navigation section.
  • <main> Defines the main content of the page.
  • <section> Represents standalone and independent content.
  • <footer> Defines the footer of the page or a section.

    Topic 2: Content Structuring

  • <section> Defines a logical or thematic section. It's used to group related content together.
  • <aside> Represents secondary or related content.
  • <figure> Used to insert multimedia content, such as images.
  • <figcaption> Provides a caption for elements within <figure>
  • <details> Creates a details element that can be expanded or collapsed.
  • <summary> Summary text for the details.
