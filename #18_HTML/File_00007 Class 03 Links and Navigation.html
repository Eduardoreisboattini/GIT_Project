Class 3: Links and Navigation

  1. Creating Hyperlinks (Anchor Tags)

    2. An anchor tag is used to create hyperlinks. It is defined using the `` element.

    <a href="https://www.example.com">Visit Example</a>
  2. Linking to External Websites

    3. To link to an external website, use the full URL as the `href` attribute of the anchor tag.

    <a href="https://www.external-site.com">Go to External Site</a>
  3. Linking to Internal Pages

    4. For internal pages, use relative URLs in the `href` attribute.

    <a href="/about">About Us</a>
  4. Creating Email Links

    5. To create an email link, use the `mailto` scheme in the `href` attribute.

    <a href="mailto:info@example.com">Send Email</a>
  5. Adding Image Links

    6. You can also create image links by placing an `` tag inside an anchor tag.

    <a href="https://www.example.com"><img src="image.jpg" alt="Image Link"></a>
