Class 7: HTML Tables

  1. Creating a Basic Table

    2. Tables are created using the `

    ` element, and rows are defined using `` elements.

    <table> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> </tr> <tr> <td>Row 2, Cell 1</td> <td>Row 2, Cell 2</td> </tr> </table>
  2. Table Headers and Data Cells

    3. Table headers are defined using `

    ` elements, and data cells are defined using `` elements.

    <table> <tr> <th>Header 1</th> <th>Header 2</th> </tr> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> </tr> </table>
  3. Spanning Rows and Columns

    4. Row and column spanning can be achieved using the `rowspan` and `colspan` attributes.

    <td rowspan="2">Spanned Cell</td> <td colspan="2">Spanned Cells</td>
  4. Styling Tables with CSS

    5. Tables can be styled using CSS to improve their appearance.

    <style> table { border-collapse: collapse; width: 100%; } th, td { border: 1px solid black; padding: 8px; text-align: left; } </style>
  5. Creating a Responsive Table

    6. For responsive tables, you can wrap the table in a `

    ` element and apply CSS for horizontal scrolling.

    <div style="overflow-x: auto;"> <table> </table> </div>
